CADStat: Statistical Tools for Causal Analysis

Welcome to CADStat

CADStat is a set of graphical and statistical tools that are useful in causal analysis. Use of these tools to help inform causal analyses is described more fully in the Causal Assessment Diagnosis Decision Information System (CADDIS).

CADStat is built on R, a powerful, freely-available statistical software package,

and JGR, a Java graphical user interface for R.
